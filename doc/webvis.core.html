build-web

(build-web spider url worker-count max-depth)

Begins the process of crawling sites, starting at the specified root url. The crawler will not crawl beyond the specified max depth. Once there are no more urls to crawl, it will terminate.

clear!

(clear!)

Clears the web graph. Disclaimer: This function is currently broken and should not be used. The synchornizedGraph will not clear synchornized node references, causing some errors when rebuilding the web.

freeze!

(freeze! spider)

Kills all workers, freezing the spider until more workers are added.

kill!

(kill! spider)(kill! spider wid)

Kills the spider, terminating any running threads associated with it and closing all channels. If a worker id is passed in, it will kill the worker with that id instead.
